# Soroban Companion Project

This is a practice project for learning Soroban smart contract development on the Stellar blockchain.

## Project Structure

This repository uses the recommended structure for a Soroban project:
```text
.
├── contracts
│   └── companion
│       ├── src
│       │   ├── lib.rs
│       │   └── test.rs
│       └── Cargo.toml
├── Cargo.toml
└── README.md
```
